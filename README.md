# 🔐 The Murdok Secure Pass

Un generador de contraseñas interactivo y seguro desarrollado con tecnologías FrontEnd puras. Esta herramienta permite a los usuarios crear contraseñas altamente seguras y fáciles de recordar basándose en reglas mnemotécnicas (frases, años y nombres personales).

## 🚀 Características

- **Generación Personalizada:** Crea contraseñas únicas basadas en inputs del usuario.
- **Validación en Tiempo Real:** Algoritmos en JS que verifican la correcta introducción de los datos.
- **Privacidad Total:** Al ser un proyecto 100% Client-Side (FrontEnd), no hay bases de datos. Ningún dato ingresado es recopilado ni almacenado.
- **Comprobador Integrado:** Integra un `iframe` de *Password Monster* para verificar la fortaleza de la contraseña generada en tiempo real.
- **UI/UX Moderna:** Animaciones CSS, transiciones suaves y diseño 100% Responsive.

## 💻 Tecnologías Utilizadas

- **HTML5:** Estructura semántica.
- **CSS3:** Flexbox, CSS Grid, Animaciones Keyframes y Media Queries.
- **JavaScript (Vanilla):** Manipulación del DOM, validación de formularios y lógica de concatenación.
